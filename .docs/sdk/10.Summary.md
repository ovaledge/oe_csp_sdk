# 10. Summary

Before starting new connector development, confirm contract details in the [Connector Interfaces Reference](../CONNECTOR_INTERFACE_REFERENCE.md).

**Prerequisites**
1. JDK 21
2. Maven 3.8+
3. `oe_csp_sdk_core` built and installed, or available in your Maven repository

**Connector (single module implementing)**
1. `AppsConnector`
2. `MetadataService`
3. `QueryService`
4. SPI registration in `META-INF/services/`
5. Optional **quick** app for standalone testing

**Build**
1. Run `mvn clean install`, or
2. Use `./build-csp-sdk.sh` to produce the assembly JAR

**Run**
1. Use `csp-api` for a unified API and UI serving all connectors, or
2. Run a connector's **quick** app to start only that connector

**Adding a New Connector**
1. Create a new module using the Maven archetype
2. Implement the required connector contracts (see [Section 5](#5-what-you-will-develop))
3. Add the SPI registration file
4. Add the module and dependencies to: Parent POM, `csp-api`, and assembly module
5. Build and test
6. (Optional) Submit a PR for public catalog inclusion
